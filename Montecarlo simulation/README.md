Monte Carlo Portfolio Simulation

This project implements a Monte Carlo simulation framework for portfolio analysis using a Geometric Brownian Motion (GBM) model.

It simulates thousands of possible future price paths for a multi-asset portfolio and evaluates performance and risk metrics.

---

📊 Features

- Multi-asset portfolio simulation
- Correlated returns via Cholesky decomposition
- Monte Carlo simulation (2000+ scenarios)
- Key metrics:
  - CAGR (Compound Annual Growth Rate)
  - Annualized Volatility
  - Sharpe Ratio
  - Value at Risk (VaR 95%)
  - Expected Shortfall (ES 95%)
  - Maximum Drawdown
- Distribution visualization of terminal returns

---

⚙️ Methodology

The simulation is based on:
- Geometric Brownian Motion (GBM)
- Cholesky Decomposition
