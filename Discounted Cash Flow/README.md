📊 DCF Valuation Tool
📖 Overview
This project implements a Discounted Cash Flow (DCF) model in Python to estimate the intrinsic value of publicly traded companies.
The model uses financial data retrieved from Yahoo Finance and projects future Free Cash Flows (FCF), discounting them to present value using a specified discount rate (WACC).
The output provides an estimate of fair value per share along with a simple investment indication: BUY, SELL, or HOLD.



⚙️ Features
Automatic financial data retrieval using yfinance
Historical Free Cash Flow (FCF) analysis
CAGR (Compound Annual Growth Rate) calculation
Future cash flow projection
Discounting using WACC
Terminal value calculation
Fair value per share estimation
Investment recommendation (BUY / SELL / HOLD)


⚠️ Disclaimer
This model is simplified and intended for educational purposes only.
Do not use this tool as the sole basis for investment decisions.
